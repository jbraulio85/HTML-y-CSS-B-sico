Proyectos

Proyecto 1:

CRUD JAVA a nivel de consola

Este proyecto, desarrollado en Java, sigue el patrón de diseño Modelo-Vista-Controlador (MVC) para simular la capa de vista se utiliza impresiones a nivel de consola. Este tipo de proyectos es muy versilatil y lo convierte en una herramienta ideal para la iniciación en programación y la aplicación de buenas prácticas de desarrollo. Adheriéndose a nomenclaturas internacionales y enfocándose en la lógica de programación, este proyecto proporciona una base sólida para la comprensión y aplicación de algoritmos y diagramas de flujo. Este enfoque profesional y estructurado sirve como un recurso valioso para aquellos que deseen adquirir habilidades sólidas en desarrollo de software desde sus etapas iniciales

Proyecto 2:

Pokedex React-Native

El proyecto "Pokedex" es una aplicación móvil desarrollada con React Native y Expo que consume una API pública de Pokémon. La aplicación ofrece una lista paginada de Pokémon con detalles como imágenes, tipos, estadísticas y habilidades, permitiendo a los usuarios explorar la información detallada de cada Pokémon. Con una interfaz amigable y funciones como búsqueda, filtrado por tipo y gestión de estado efectiva, la aplicación proporciona una experiencia de usuario intuitiva. Tecnologías clave incluyen React Native, Expo y la API de Pokémon. Futuras contribuciones podrían enfocarse en agregar funcionalidades como la capacidad de guardar Pokémon favoritos, mejorar el diseño y la optimización del rendimiento. El proyecto está abierto a la colaboración de la comunidad para enriquecer la aplicación y hacerla aún más completa y atractiva para los amantes de Pokémon.
